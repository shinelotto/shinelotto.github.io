# 🎰 彩票分析系统

一个基于 Flask 的 Web 应用，专门用于分析中国福利彩票双色球和体育彩票大乐透的历史数据。系统提供丰富的走势分析、统计图表和数据可视化功能，帮助用户更好地理解彩票号码的分布规律。

## ✨ 主要特性

### 📈 核心分析功能
- **双色球（SSQ）分析**
  - 红球基本走势分析
  - 蓝球走势统计
  - 冷温热号码分析
  - 号码遗漏期数统计
  - 区间分布分析

- **大乐透（DLT）分析**
  - 前区号码走势
  - 后区号码统计
  - 遗漏数据分析
  - 号码热度分析

### 📊 数据统计指标
- **基本统计**：和值、跨度、AC值
- **区间分析**：大小比、奇偶比、质合比
- **号码关系**：连号分析、同尾号分析
- **012路分析**：0路、1路、2路号码分布
- **冷温热状态**：基于遗漏期数的号码热度分类

### 🎨 可视化功能
- 走势图表展示
- 号码热度颜色标注
- 统计图表可视化
- 响应式Web界面
